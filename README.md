# clepsydra
15 min cyclic timer for android. Ultra-minimalist UI and functionality. Geek health helper.
We all spend hours hunched over a computer screen. To stay physically fit, you need to take regular breaks and do muscle exercises—like squats, push-ups, burpees, and eye exercises. Move your body every 15 minutes—the timer will remind you. No other functions—just a regular beep.
1. Install the app and launch it.
2. Each cycle begins with a beep.
3. Pull down the system tray—there will be a countdown timer and a stop button.
That's it.
If you want to change the timer length or sound, recompile the source code with your own settings.

Why I made this app? Because the simplest loop timer on Google Play consists of 12 settings.
